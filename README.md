# 📅 Daily Flow Dashboard

A personal daily dashboard built with React, TypeScript and Vite — created as a side project using AI-assisted (vibe coding) development with Claude Code.

## Features

- ✅ Daily reminder & task overview
- 💸 Add and track daily expenses
- 📊 Expense history view

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS Modules

## About

This project was built as a personal productivity tool and also as an exercise in vibe coding — describing features in natural language and iterating with AI (Claude Code) to produce working components.

## Getting Started
```bash
npm install
npm run dev
```
